純數值 Number

結果：{{ numberMsg }}  typeof is {{ typeof numberMsg }}

沒有修飾符：
即使輸入的是數值，仍會被判斷為文字。

加入修飾符：
input type 設定為 text，v-model 加上 .number 修飾符。
輸入純數值時，可以被判斷為數字。

然而，這樣的做法需要注意一些特性：

  1. 不會限制只能輸入數字。
  2. 若開頭輸入的是文字，輸入的值會進到 Vue 資料中，並被判斷為文字 (string)。
  3. 若開頭輸入的是數字，輸入的值會進到 Vue 資料中，並被判斷為數字 (number)。
    但是，若後續接續文字，在畫面上雖然能鍵入文字，但文字的部份不會進到 Vue 資料中，而判斷也仍然為數字 (number)，這可能會造成奇怪的體驗。

沒有修飾符，input type 設定為 number。
v-model 不需要再另外加上 .number 修飾符。
.number 修飾符會在輸入框有 type="number" 時自動啟用。(官方說明)
